# Security Policy

## Supported Versions

Any security update will be delivered on current version. If particularly relevant and if it is reasonably feasible, older version might get the fixes too.

| Version | Supported          |
| ------- | ------------------ |
| 1.6.x   | :white_check_mark: |
| < 1.6.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability either open an issue here on GitHub or consider contacting the author directly if you think that's the case.
